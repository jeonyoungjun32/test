==DB Project 시 고려할 사항, 순서
1. VO 객체 생성 == table 1 : 1 mapping(테이블 당 각 한개의 VO)
2. interface 객체 생성 == CRUD를 포함해서 필요한 형태로 구성
3. DAO(데이터베이스 연결객체를 만든다)
   1. 드라이버 ClassName
   2. DB 물리적 주소 url
   3. DB 사용자 계정
   4. DB 사용자계정 패스워드
   Connection 객체를 돌려 받는다. 
4. interface 객체의 구현체를 만든다.(인터페이스명.impl.java)
5. 필요한 작업을 수행하는 App객체를 만든다.
6. java에서는 main()에 실행할 App를 조립한다.
